COVID-19 Forecasting

This page provides a resource for analytics concerning the spread of COVID-19 in the United States. The number of new daily infections are predicted using a combination of an SEIR epidemiological model and a nearest neighbor heuristic measure of infection. Toggle the dashboard below to view the forecasted number of daily infections in your county. Visit the methods section to learn more or the contact section if you have any questions. Work was done independently while a student at New York University
